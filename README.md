# OpenTraffic-IQC

Public delivery repository for traffic-scene image quality classification.

## Delivery Contents

This repository includes:

1. shared library delivery in `artifacts/`
2. validation summaries in `validation/`
3. public manifest in `manifests/`
4. checksum file in `checksums.txt`

## Quick Start

1. Verify the delivered files with `checksums.txt`.
2. Review `manifests/public_manifest.json`.
3. Review `validation/public_validation_summary.json`.
4. Integrate `artifacts/libimage_quality_classification.so` into the target Linux runtime.
5. Use the validation summary as the baseline acceptance record.

See `USAGE.md` for the delivery usage steps.
